# cz-git 简易版教程
> [cz-git](https://github.com/Zhengqbbb/cz-git)

## yarn 项目:
> 使用 commitizen 命令行工具

### 1. 请确保你已经全局安装 commitizen 命令行工具
```bash
npm install -g commitizen
``` 

### 2. 克隆项目并下载依赖
```bash
git clone -b commitizen-yarn-zh https://github.com/Zhengqbbb/czgit-playground.git
cd czgit-playground
yarn
```

### 3. 试试看！
```bash
git add .
cz
```
